Change woocommerce price field zero to Free text.

== Description ==

Zero to free text plugin is help you change 0 value to "Free!" text. You must have already installed woocommerce.

= Features =

* Easy to use.
* Compatibility with many other plugins.

== Installation ==

1. Upload the `zero-to-free-text` folder to the `/wp-content/plugins/` directory or install it from the plugin directory via your Plugins dash.
2. Activate the Zero to free text plugin through the 'Plugins' menu in WordPress
